# Checkboxes

**checkboxes.js** is a simple library and jQuery plugin to support checked and unchecked checkboxes.

## Usage

```javascript
$('#chkAll').checkboxes({ itemChild: 'chk'});
```

## Options

- `itemChild` The name of children checkboxes
